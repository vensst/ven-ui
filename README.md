# ven-ui
## 说明
本项目基于element-ui@2.15.3版本改造而来，只支持vue@2.x版本。   
[ven-ui文档](https://vensst.github.io/ven-ui/)    
[element-ui文档](https://element.eleme.cn/#/zh-CN)  

## 安装
```shell
npm install ven-ui -S
```

## 使用
```javascript
// main.js
import Vue from "vue";
import VenUi from "ven-ui";
import "ven-ui/lib/index.css"; 
Vue.use(VenUi);
```
